# GCP Essentials - Qwiklabs 실습 개요

 

Qwiklabs의 GCP Essentials Quest를 수강하며 작성한 포스팅이다.

GCP Essentials는?

Google Cloud를 처음으로 입문할 때 실습하며 GCP 사용법을 익힐 수 있는 기본적인 튜토리얼이다!

이 Quest는 클라우드 컴퓨팅이나 Google 클라우드 플랫폼에 대한 사전 지식이 필요없다!
학생들은 information technology 또는 computing background의 이해가 있고 컴퓨팅 시스템 관리에 대해 익술할 것으로 기대된다.

도움되는 사전지식
셸(Shell) 환경/명령줄(Command line) 인터페이스에 대해 알고있다면 Quest를 수행하는데 도움이 될 것이다.

 

Quest Outline

✅ Qwiklabs 및 Google Cloud Platform 둘러보기

이 실습에서는 Qwiklabs와 Google Cloud Platform Console에 액세스하고
기본 GCP 기능인 프로젝트, 리소스, IAM 사용자, 역할, 권한, API 및 Cloud Shell을 사용합니다.

 

✅ 가상 머신 만들기

 

이번 실습에서는 Google Compute Engine 가상 머신을 만드는 방법을 배우고 영역, 리전 및 머신 유형에 관해 알아봅니다.
내용을 미리 살펴보려면 짧은 동영상인 가상 머신 만들기, GCP Essentials를 확인하세요.

 

✅ Compute Engine: Qwik Start - Windows

 

Google Compute Engine을 사용하면 Google 인프라에서 가상 머신을 만들고 실행할 수 있습니다.
이 실험실에서는 Google Compute Engine에 Windows 서버 인스턴스를 만들고 RDP로 액세스하는 방법에 대해 알아봅니다. 짧은 미리보기인 Windows 서버 인스턴스 실행하기, GCP 기본사항을 시청하세요.

 

✅ Cloud Shell 및 gcloud 시작하기

 

웹을 통해 Google Cloud Platform에 호스팅된 컴퓨팅 리소스에 연결하는 방법을 학습합니다. Cloud Shell과 Cloud SDK gcloud 명령어를 사용하는 방법도 배우게 됩니다. 미리 살펴보려면 짧은 동영상인 Cloud Shell 시작하기, GCP 기본사항을 시청하세요.

 

✅ Kubernetes Engine: Qwik Start

Google Kubernetes Engine에서는 Google 인프라를 사용하여 컨테이너식 애플리케이션을 배포, 관리 및 확장할 수 있는 관리형 환경을 제공합니다. 이 실습에서는 Kubernetes Engine으로 컨테이너식 애플리케이션을 배포하는 방법을 알아봅니다. 짧은 동영상 Kubernetes Engine을 사용하여 컨테이너식 앱 관리에서 자세히 알아보세요.

 

✅ 네트워크 및 HTTP 부하 분산기 설정하기

이번 실습에서는 Google Compute Engine 가상 머신에서 실행되는 사용자 애플리케이션을 위한 네트워크 부하 분산기 및 HTTP 부하 분산기를 설정하는 방법을 배웁니다. 간단한 네트워크 및 HTTP 부하 분산기 설정하기, GCP Essentials 미리보기를 확인하세요.

 

✅Google Cloud Essentials: Challenge Lab

Challenge Lab에서는 .GCP Essentials 퀘스트에 있는 lab의 기술과 지식을 테스트합니다.
이 랩을 시도하기 전에 랩의 내용을 숙지해야 합니다.
