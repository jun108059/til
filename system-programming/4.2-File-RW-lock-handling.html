# System Programming

# EX4.2 - File RW lock handling

ATM 예제 코드

  • ATM command (c/r/d/q)

  • c : create

  • r : read

  • d : deposit

  • q : quit

  • w : withdraw

  • t : transfer

    Withdraw

    1. Deposit과 유사하게 구현
    2. writer's lock 이용
    3. balance 값을 요청하는 출금 금액만큼 감소 후 unlock

    Transfer

    1. withdraw + deposit 순으로 진행
    2. 출금 계좌 및 예금 계좌에 writer's lock
    3. 출금 계좌 (송금하는 계좌)에서 withdraw 후 입금 계좌 (송금받는 계좌)에서 deposit
    4. 모든 작업 후 unlock
